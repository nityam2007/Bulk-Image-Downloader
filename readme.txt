Easily download and import multiple images from external URLs to your media library.

== Description ==

Bulk Image Downloader is a powerful WordPress plugin that allows you to download and import multiple images from external URLs directly to your media library. It provides a convenient way to bulk download images and use them in your WordPress posts, pages, or WooCommerce products.

== Installation ==

1. Upload the `bulk-image-downloader` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Bulk Image Downloader' menu in the WordPress admin dashboard.
4. Enter the image URLs in the provided textarea, with each URL on a new line.
5. Click the 'Download Images' button to initiate the bulk download process.
6. The downloaded images will be saved in your WordPress media library and can be used in your posts, pages, or products.

== Frequently Asked Questions ==

= Can I download images from any URL? =

You can download images from most URLs as long as they are publicly accessible and in a supported image format (JPEG, PNG, GIF).

= Can I use the downloaded images in WooCommerce products? =

Yes, once the images are downloaded and added to your media library, you can easily set them as featured images or gallery images for your WooCommerce products.
